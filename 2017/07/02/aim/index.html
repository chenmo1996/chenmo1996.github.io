做一个出色的程序员，摄影师，舞者

2017-07-02

人的开心来自与对未来的希望，痛苦来自对未来的迷茫。

今天想清楚了自己想要什么，真是值得纪念的一天。

今天参观了高鹏大佬的出租屋，略显脏乱又颇感温馨，深刻庆幸着自己有一颗不希望浪费生命的心。做一个平凡的人，有自己喜欢的事情，也已经很幸福了。

做一个程序员，希望你，起于代码却不止于代码，全情投入到这一份带给你生活经济基础和快乐的工作中去，不断学习和进步，让自己综合更加优秀

作为一个摄影师，希望你保持创意，勿忘初心。将你对生活的理解，用相机做画笔，统统表达到你的作品中去。

作为一个舞者，记得享受音乐，享受友谊，懂得感恩，还要努力！

有这么多想法，真的是很贪心，但是停不下来啊！！！加油！！！
